Web Development Today

Front-End: Which has to do with document structuring using HTML5 and document styling using CSS3. It is also known as the client-side because it provides platform for interaction between the clients (web browsers like IE and Firefox) and users.

Back-End: Which has to do with interaction between web browsers and servers, as well as writing data and retrieving it from databases. A lot of back-end languages were in use today including PHP, PYTHON, JAVA, etc. Database used is popularly mySQL.
